# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-01

### Added
- **Core Timer**: Precision timer with inspection, stackmat support, and keyboard shortcuts.
- **Scrambles**: Support for all WCA puzzles (3x3, 4x4, 5x5, 6x6, 7x7, Pyraminx, Megaminx, Skewb, Square-1, Clock).
- **Session Management**: Create, rename, and delete sessions. Merge guest sessions into account.
- **Cloud Sync**: Seamless synchronization across devices using Supabase.
- **Analytics**: Detailed statistics including Best/Current Ao5, Ao12, Ao100, Mean, Std Dev, and more.
- **Interactive Graphs**: Visualize solve trends over time.
- **Customization**: Multiple themes (Dark, Light, AMOLED, Nord, etc.) and UI settings.
- **PWA Support**: Installable as a native-like app on mobile and desktop.
- **Offline Mode**: Full functionality without internet connection.
- **Import/Export**: Support for CSTimer and JSON data import/export.
- **Social**: Share your results with a generated image.
- **Monetization**: AdSense integration and "Support" page with Ko-fi link.
